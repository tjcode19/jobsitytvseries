# jobsitytvseries

Jobsity TV Series Tes .

## Getting Started

Flutter application for listing TV Series .

To run this app:

- Clone from repo with
```
git clone https://github.com/tjcode19/jobsitytvseries.git
```

- Install all dependencies
```
flutter pub get
```

- Run the app
```
flutter run
```
